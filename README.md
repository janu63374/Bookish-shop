# 📚 Bookish Shop

Bookish Shop is a full-stack online bookstore web application developed using PHP, MySQL, HTML, CSS, and JavaScript.

## 🚀 Features
- User login system
- Book browsing
- Shopping cart & wishlist
- Responsive design
- Database integration

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## 💻 How to Run
1. Move project to XAMPP htdocs folder
2. Start Apache and MySQL
3. Import database in phpMyAdmin
4. Open:
http://localhost/Bookshop
