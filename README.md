# 📚 Bookish Shop

Bookish Shop is a full-stack online bookstore web application developed using PHP, MySQL, HTML, CSS, and JavaScript.

## 🚀 Features
- User login system
- Book browsing
- Shopping cart & wishlist
- Responsive design
- Database integration

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
